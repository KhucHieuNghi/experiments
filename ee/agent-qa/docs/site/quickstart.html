Quickstart

Install ETUS and run your first AI-powered test in under five minutes.

Prerequisites

  • Node.js >= 24
  • Docker (optional, required only for sandboxed hooks)

Installation

Install the CLI

npm install -D etus-agent

Optionally install the subscription auth package for team features:

npm install -D @etus/agent-subscription-auth

Install browsers

npx etus-agent install-browsers --all

Downloads Chromium, Firefox, and WebKit for Playwright-based testing.

Initialize your project

npx etus-agent init

Creates the etus-agent.config.yaml configuration file and a sample test.

Verify your setup

npx etus-agent doctor

Checks Node version, browser availability, Docker access, and config validity.

Generated config file

After init, your project root will contain:

# etus-agent.config.yaml
version: 1

llm:
  provider: openai
  model: gpt-4o
  apiKey: ${OPENAI_API_KEY}

use:
  browser: chromium
  baseUrl: http://localhost:3000
  headless: true

services:
  cache:
    dir: .etus-agent/cache
    ttl: 86400
  memory:
    dir: etus-agent-memory
  dashboard:
    port: 4100
Tip
Create a etus-agent.local.yaml file for machine-specific overrides (API keys, ports). This file should be in your .gitignore.

Launch the Dashboard

npx etus-agent dashboard --open

Opens the local dashboard at http://localhost:4100. From here you can connect your LLM, browse tests, and trigger runs interactively.

Run your first test

npx etus-agent run tests/example-pass.yaml

ETUS will launch a browser, interpret the natural language steps, and report results to the terminal and dashboard.

CI usage

In a CI pipeline, run tests headlessly with a single command:

npx etus-agent run tests/example-pass.yaml

Set ETUS_AGENT_HEADLESS=true and provide your LLM API key via environment variables. See Environment Variables for the full list.
