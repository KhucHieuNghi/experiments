Configuration

ETUS uses a layered, file-backed configuration system. Settings resolve from global config down through suites, tests, and CLI flags.

Resolution order

Configuration is resolved with the following precedence (highest wins):

  1. CLI flags (e.g., --log-level debug)
  2. Environment variables (ETUS_AGENT_*)
  3. Per-test use overrides
  4. Suite-level use overrides
  5. Local override file (etus-agent.local.yaml)
  6. Global config file (etus-agent.config.yaml)

Configuration concepts

Global Config

The etus-agent.config.yaml file at project root. Defines LLM provider, browser settings, services, and defaults.

Local Override

The etus-agent.local.yaml file. Machine-specific overrides (API keys, ports) that stay out of version control.

Test Overrides

Per-test use block. Override browser, base URL, or other settings for a single test.

Suite Overrides

Suite-level use block applies to all tests in the suite unless a test overrides further.

Variables

Template variables with {{var}} syntax. Secrets use {{secret:NAME}} for redaction in output.

Auth Store

Credentials stored at ~/.etus-agent/auth.json. Managed by etus-agent auth commands.

Typical project structure

my-project/
  etus-agent.config.yaml      # Global config (committed)
  etus-agent.local.yaml       # Local overrides (gitignored)
  tests/
    login.yaml                # Test files
    checkout.yaml
  suites/
    smoke.yaml                # Suite definitions
    regression.yaml
  hooks/
    hooks.yaml                # Hook definitions
    scripts/
      seed-db.mjs            # Hook scripts
  etus-agent-memory/          # Memory observations
    products/
    suites/
    tests/
  .etus-agent/                # Runtime artifacts (gitignored)
    cache/                    # Action plan cache
    runs/                     # Run history

Full config reference

# etus-agent.config.yaml
version: 1

llm:
  provider: openai           # openai | anthropic | azure | custom
  model: gpt-4o
  apiKey: ${OPENAI_API_KEY}
  temperature: 0.2
  maxTokens: 4096

use:
  browser: chromium          # chromium | firefox | webkit
  baseUrl: http://localhost:3000
  headless: true
  viewport: { width: 1280, height: 720 }
  video: false
  timeout: 30000

services:
  cache:
    dir: .etus-agent/cache
    ttl: 86400               # Seconds
  memory:
    dir: etus-agent-memory
  dashboard:
    port: 4100
  mcp:
    port: 4200

variables:
  env: staging
  region: us-east-1
Info
Use npx etus-agent config to print the resolved configuration for debugging. See CLI Commands for details.
