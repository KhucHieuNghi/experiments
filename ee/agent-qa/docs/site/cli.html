CLI Commands

Complete reference for the etus-agent command-line interface.

Commands

CommandDescription
etus-agent run <path>Run a test or suite YAML file
etus-agent initInitialize a new project with config and sample test
etus-agent dashboardStart the local dashboard server
etus-agent mcpStart the MCP (Model Context Protocol) server
etus-agent doctorCheck environment: Node, browsers, Docker, config
etus-agent auth setManually store auth credentials
etus-agent auth testValidate a stored auth state
etus-agent auth statusList all stored auth states
etus-agent auth loginInteractively capture browser auth state
etus-agent auth logoutRemove a stored auth state
etus-agent install-browsersDownload Playwright browser binaries
etus-agent configPrint the resolved configuration
etus-agent validateValidate test, suite, and hook YAML files
etus-agent cache purgePurge cached action plans
etus-agent clean-memoryRemove deprecated memory observations

Global options

OptionDescription
--config <path>Path to config file (default: etus-agent.config.yaml)
--log-level <level>Set log level: silent, error, warn, info, debug
--verboseShorthand for --log-level debug
--quietShorthand for --log-level silent

etus-agent run

Execute a test or suite file.

npx etus-agent run <path> [options]

Options

OptionDescription
--no-cacheSkip cache reads (still writes new entries)
--browser <name>Override browser: chromium, firefox, webkit
--headlessRun in headless mode (default: from config)
--base-url <url>Override base URL
--timeout <ms>Override action timeout

Examples

# Run a single test
npx etus-agent run tests/login.yaml

# Run a suite
npx etus-agent run suites/smoke.yaml

# Run with debug logging and no cache
npx etus-agent run tests/checkout.yaml --verbose --no-cache

# Run in Firefox
npx etus-agent run tests/login.yaml --browser firefox

etus-agent dashboard

Start the local dashboard web UI.

npx etus-agent dashboard [options]

Options

OptionDescription
--openOpen the dashboard in your default browser
--port <number>Override dashboard port (default: 4100)

etus-agent mcp

Start the MCP server for AI tool integration.

npx etus-agent mcp [options]

Options

OptionDescription
--port <number>Override MCP port (default: 4200)
--transport <type>Transport: http or stdio

etus-agent install-browsers

Download Playwright browser binaries.

npx etus-agent install-browsers [options]

Options

OptionDescription
--allInstall all supported browsers
--chromiumInstall Chromium only
--firefoxInstall Firefox only
--webkitInstall WebKit only

etus-agent cache

Manage the action plan cache.

npx etus-agent cache purge [options]

Options

OptionDescription
--allPurge all cached entries
--prefix <string>Purge entries matching prefix
