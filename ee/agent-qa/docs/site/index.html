What is ETUS

An AI-powered QA agent that writes, runs, and improves end-to-end tests using natural language. Bring your own LLM, describe what to test, and let ETUS handle the rest.

Quickstart

Install ETUS and run your first test in under five minutes.

Configuration

File-backed config with layered resolution from global to per-test.

Memory

Persistent observations that make your agent smarter over time.

Caching

Action plan caching for 5x faster runs and 3x fewer tokens.

Tests

Write natural language test steps in simple YAML files.

Web Testing

Playwright-based browser automation with smart waits and accessibility.

Auth State

Capture login once, reuse by name, keep credentials out of artifacts.

Suites

Group tests with shared hooks and configuration overrides.

Hooks

Run sandboxed scripts for setup, teardown, and data generation.

How it works

Write tests in natural language

Describe user journeys as plain-English steps in YAML. No page objects, no selectors, no brittle locator chains.

Bring your own LLM

Connect any OpenAI-compatible provider. ETUS orchestrates the model but never locks you into a single vendor.

Run tests

ETUS launches a browser, interprets each step, interacts with the page, and validates outcomes autonomously.

Generate memory

After each run the agent curates observations about your product into a persistent memory layer.

Improve the next run

Memory feeds back into planning so the agent gets faster, more accurate, and more context-aware with every execution. Grow a product-aware QA agent over time.
